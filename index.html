🔢 압축 슬라이더 : 1~14까지 선택. AI 또는 특수분석 모드에서 사용할 숫자 개수입니다. 값이 작을수록 상위 점수의 숫자만 사용합니다. (기본 9)

🤖 AI / 🎯 특수분석 모드 : AI는 패턴(이웃번호, 갭)을 분석하여 점수를 매기고, 특수분석은 생존수(최근 출현 빈도 상위 20개) 중에서 최근 10회 출현 빈도와 15회 미출현 가중치 등으로 점수를 매겨 숫자를 선정합니다.

간격(주) / 회차(개) : 기준 회차로부터 몇 주 간격으로 몇 개의 회차를 분석할지 설정합니다. 예: 간격 2주, 회차 10개 → 기준 회차 직전부터 2주 간격으로 10개 회차를 수집합니다.

교환 개수 : 트레이드에서 교환할 숫자 개수 (1~8개).

🔄 트레이드 : AI 집합과 특수분석 집합 간에 점수가 낮은 번호와 높은 번호를 교환합니다. 트레이드 후에는 '추천'이라는 통합14수가 생성되고, 유실된 번호가 표시됩니다. 유실된 번호 중 트레이드 전 교집합이었던 번호는 자동으로 고정수에 등록됩니다.

1조합 / 5조합 / 10조합 : 현재 선택된 모드(AI 또는 특수분석)의 압축 숫자 풀에서 조합을 생성합니다. 고정수가 있으면 1~2개 포함되고, 고정수가 없으면 합계, 홀짝, 연속번호 등의 기본 필터가 적용됩니다. (1조합 버튼은 주황색, 5/10조합은 청록색)

통합5 / 통합10 : 트레이드 후 생성된 통합14수에서 조합을 생성합니다. (트레이드 후에만 사용 가능)

B바구니 : 유실번호 + 전체교집합 + 통합미사용 번호를 조합합니다. 유실된 당첨번호는 초록색으로 표시됩니다.

🔒 고정수 설정 : '🔒 고정' 버튼(트레이드 옆)을 누르면 고정수 선택 팝업이 열립니다. 최대 14개까지 선택 가능하며, 조합 생성 시 1~2개가 랜덤 포함됩니다. '🔷 교집합 5~6위 고정' 버튼은 트레이드 후 교집합이 있을 때 점수 순으로 5~6위에 해당하는 번호를 고정수에 등록합니다. (교집합이 5개 미만이면 동작하지 않음)

📊 당첨확인 : 저장된 조합과 입력한 당첨번호(또는 해당 회차의 실제 당첨번호)를 비교하여 등수를 표시합니다. '저장' 버튼으로 번호를 저장하면 상태창에 자동 반영됩니다.

📋 생성조합 : 최근 생성된 조합 목록을 보여줍니다. 각 조합 옆의 '저장' 버튼으로 보관함에 저장할 수 있습니다.

🎮 게임 : 터치 게임(공 튕기며 숫자 선택)을 즐길 수 있습니다.

초기화 / 새로고침 : '초기화'는 저장된 조합을 모두 삭제합니다. '새로고침'은 페이지를 다시 로드합니다.

🌐 네이버 : 네이버로 이동합니다.

📦 보관함 : 저장한 조합들을 확인하고 관리할 수 있습니다.

⏳ 추가 : 준비 중인 기능입니다.
